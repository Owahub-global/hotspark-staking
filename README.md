# 🔥 HotSpark Staking

Professional staking contract for HotSpark (HOT) tokens on BSC.

## ✨ Features

- **Stake HOT** - Earn rewards in HOT
- **Fixed APR** - Owner sets reward rate
- **Time-based** - Rewards calculated per second
- **Secure** - ReentrancyGuard, Pausable, SafeERC20
- **Gas Efficient** - Uses reward-per-token accounting
- **No Inflation** - Rewards funded separately

## 📋 Prerequisites

- Node.js v18+
- npm or yarn
- BSC Testnet BNB
- HotSpark Token deployed
